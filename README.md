# Vim Cheat Sheet

This page is for vim cheat sheet



### Cursor movement

​										k (move cursor up)

h (move cursor left)									l (move cursor right)

​										j (move cursor down)



0 (jump to the start of the line)	<------------>	$ (jump to the end of the line)



### 삽입 모드

i 현재 커서 앞에 삽입

​     

### 선택 모드

v 비주얼 모드 



### 잘라내기와 붙여넣기

yy 한 행 복사하기

p 커서 뒤에 붙여넣기

P 커서 앞에 붙여넣기



### 나가기

:w 저장하기

:q 나가기



### 여러 파일 작업

:vs 새 버퍼에 파일 열고 좌우로 창 분할
